# ATLAS Phase 1 Constitution

These principles are frozen for Phase 1. Do not revisit during implementation.

1. **The system preserves evidence; it does not interpret markets.**
2. **Every recorded event is immutable.**
3. **Replay emits the same event interface as live capture.**
4. **Adapters are exchange-specific; the core is exchange-agnostic.**
5. **All downstream systems consume evidence—they never modify it.**
6. **Every archive is auditable, versioned, and reproducible.**
7. **The Event Bus is the single integration point for live and replay data.**
8. **Storage is pluggable; JSONL is only the first implementation.**

The recorder is the custodian of evidence, not an interpreter of markets.
